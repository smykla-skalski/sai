# Skills

Collection of skills for agentic programming workflows.

## Overview

Skills are specialized capabilities that agents can invoke to perform common development tasks:

- Code review automation
- Documentation generation
- PR management
- Project maintenance
- Development workflow optimization

## Usage

Skills integrate with Claude Code and other agentic tools to streamline development workflows.

## Structure

Each skill is self-contained with:

- Prompt instructions
- Tool specifications
- Usage examples
- Configuration options
